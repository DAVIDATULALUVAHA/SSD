Learning Outcomes of Secure Software Development Module

In this module i will be able to:

  1. Cultivate an appreciation of the basic philosophies of secure development methodologies.
  2. Explore how to carry out analysis, program design, system development and testing critical for programming.
  3. Exhibit appreciation of the basic principles of architecture, as well as old and modern
  4. Software Development Life Cycle (SDLC) models, like TOGAF and Agile.
  5. Gain capability to practice the skills acquired in a team environment, exhibit how to manage conflicts and how to make compromises,
  6. To reflect on and evaluate my own personal development.

On conclusion of this module I will be able to:

  1. Isolate and deal with security risks as part of a software development project.
  2. Critically analyze system development difficulties and decide on solutions to the problems.
  3. Design and develop a system and to produce a solution that is up to the standard specified.
  4. Steadily develop and implement the skills required to be effective member of a development team.
